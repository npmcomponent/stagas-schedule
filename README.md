
# schedule

Weekly Schedule calendar UI component.

## Install

`component-install stagas/schedule`

## Features

- Evented.
- Weekly-type calendar.
- Multiple layers.
- Variable days view.
- Draggable and resizable event items.
- Purposeless.
- Themeless.
